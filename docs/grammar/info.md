# MIL Language Grammar

## Language Structure

```
Program
  └─ Statements (kill, let, assignment, print)
      └─ Expressions
          └─ Comparison operators (<, >, <=, >=, ==, !=)
              └─ Arithmetic operators (+, -, *, /)
                  └─ Factors (literals, identifiers, parentheses)
```
